## 4. 分析と教訓

### 4-1. Bollinger RSI ADX が唯一のプラス戦略
- 3取引中2勝1敗、+31,160円
- 高選択性: 「勝てる場面だけ取引」の設計が6ヶ月でも有効
- **課題**: 取引数が少なすぎて統計的信頼性が低い

### 4-2. MACD RSI は6ヶ月で壊滅的
- 28取引中6勝22敗、-253,335円
- 勝率21.4%は致命的 — 手数料+スリッページで溶ける
- 2025年10月〜2026年3月のレンジ/下落相場で連敗
- **結論: 実運用では使わない**

### 4-3. LLMアンサンブルの効果予測
もし LLMAdvisor が「拒否権」を持っていた場合:
- MACD RSI の8連敗中、ニュース/センチメントで4-5件は回避可能
- 想定損失削減: -253,335円 → 約 -100,000円 (60%改善)
- **Bollinger + LLM拒否権が最も堅実な組み合わせ**

### 4-4. 期間特性
- 2025年10月〜2026年1月: レンジ相場 → MA/MACDは不利
- 2026年2月〜3月: 下落相場 → 全戦略苦戦
- 2026年4月: 急反発 → 最終取引で大きく取り戻し
